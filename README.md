## Суть проекта

> **Это конструктивный отрицательный результат. Опровергнутый путь ценнее недоказанного.**

Этот репозиторий документирует попытку вывести параметры Стандартной модели из симметрии группы H4 Coxeter — и **четыре формальных теоремы**, доказывающих, почему это невозможно.

## Сборка

```bash
# Coq
opam install coq.8.20.1 coq-interval coq-coquelicot
cd proofs/trinity && coq_makefile -f _CoqProject -o Makefile.coq
make -f Makefile.coq -j$(nproc)

# Python валидация
pip install mpmath numpy
python3 scripts/validators/validate_v4.py
python3 scripts/anti_numerology_gate.py

# Docker
docker run -it --rm -v $(pwd):/work coqorg/coq:8.20.1-ocaml-4.14-flambda bash
```
