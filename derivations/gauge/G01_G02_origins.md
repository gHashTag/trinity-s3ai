## 1. Что такое G01 и G02

### G01: обратная постоянная тонкой структуры

**Формула (из `validate_v4.py` и `Catalog42.v`):**

```
G01_V  =  36 · φ · e² / π
```

где φ = (1 + √5)/2 — золотое сечение, e — основание натурального логарифма, π — число Пи.

**Физический смысл:**  
G01 соответствует 1/α(m_Z) — обратной постоянной тонкой структуры на масштабе Z-бозона.

| Вычисленное значение | Экспериментальное (PDG 2024) | Отн. погрешность |
|----------------------|------------------------------|-----------------|
| 36 · φ · e² / π ≈ 137.068 | 137.035999084 | 0.024% (класс V) |

**Примечание по нормировке:**  
Это физическая 1/α(m_Z) в схеме MS-bar с электрослабым пробегом, а **не** 1/α(M_GUT). Переход от ГУТ-нормировки к физической 1/α требует пробега от Λ_H4 ~ 1.5·10¹⁶ ГэВ до m_Z ~ 91 ГэВ через однопетлевые уравнения РГГ.

### G02: константа сильного взаимодействия

**Формула (из `validate_v4.py` и `Catalog42.v`):**

```
G02  =  (√5 − 2) / 2
```

**Физический смысл:**  
G02 соответствует α_s(m_Z) — константе сильного взаимодействия на масштабе Z-бозона.

| Вычисленное значение | Экспериментальное (PDG 2024) | Отн. погрешность |
|----------------------|------------------------------|-----------------|
| (√5 − 2)/2 ≈ 0.11803 | 0.1179 | 0.11% (класс V, граничный) |

**Примечание (Coq):** Погрешность 0.11% чуть выше порога V-класса (0.1%); в `GaugeOrigins.v` доказано `|G02 − 0.1179|/0.1179 < 2/1000`.

**Связь с золотым сечением:**  
Заметим, что √5 = 2φ − 1, поэтому:

```
G02 = (√5 − 2) / 2 = ((2φ − 1) − 2) / 2 = (2φ − 3) / 2 = φ − 3/2
```

Иными словами, G02 = φ − 3/2 ≈ 1.618... − 1.5 = 0.118..., что численно совпадает с α_s(m_Z).

**Связь с H4Derivations.v:**  
В `H4Derivations.v` определено `G02_unity : 1 = 1` и `G02_unity_phi_form : powZ phi 0 = 1`. Это "единица" в смысле нормировки, а физическая константа G02 = α_s задаётся формулой `trinity_alpha_s := (sqrt 5 − 2) / 2` в `RGRunning.v`.

## 2. Число Кокстера h = 30 и его факторизация

### Число Кокстера группы H4

Группа H4 — наибольшая исключительная конечная группа отражений в 4D:
- Порядок: |H4| = 14400 = 2⁶ · 3² · 5²
- Ранг: 4
- Степени инвариантов: {2, 12, 20, 30}
- **Число Кокстера: h = 30** (наибольшая степень)

Строго доказано (Coq, `coxeter_number_factorization`):

```coq
Lemma coxeter_number_factorization : coxeter_number = (2 * 3 * 5)%nat.
Proof. reflexivity. Qed.
```

### Разложение h = 30 = 2 × 3 × 5 и калибровочные группы SM

Это разложение на три простых множителя соответствует трём калибровочным факторам Стандартной модели:

| Делитель h | Значение | Калибровочная группа |
|------------|----------|---------------------|
| h / 15 = 2 | SU(2)_L  | Слабое взаимодействие |
| h / 10 = 3 | SU(3)_C  | Сильное взаимодействие |
| h / 6  = 5 | SU(5)    | ГУТ-структура (Великое объединение) |

Строго доказано (Coq, `h30_gauge_groups`):

```coq
Lemma h30_gauge_groups :
  h30_factor_2 = 2%nat /\ h30_factor_3 = 3%nat /\ h30_factor_5 = 5%nat.
Proof. ... reflexivity. Qed.
```

**ЧЕСТНО:** Утверждение о том, что именно это разложение "объясняет" SM калибровочные группы — это физическая интерпретация числового соответствия, а не строгий теорема. Математически доказано лишь равенство 30/15=2, 30/10=3, 30/6=5.

## 3. Проекция E8 → SU(3)×SU(2)×U(1): подструктура корневой системы

### Схема вложений

Цепочка вложений, релевантная для проекции E8 → SM:

```
H4  ─→  F4  ─→  D4  ─→  A3 × A1  ─→  A2 × A2  ─→  SU(3)_C × SU(3)_L
 ↘                ↘
  A4 (ранг 4)      Aut(A4) = SU(5) ГУТ
```

Ключевые подсистемы корней H4, реализующие SM-группы:

#### 3.1. A2 × A2 → SU(3)_C × SU(3)_L (предшественник Патги-Салама)

- W(A2 × A2) ⋊ Z₂ ≅ SU(3)_C × SU(3)_L
- |W(A2)| = 6 = 3!, |W(A2 × A2)| = 36, с Z₂: 72
- Число Кокстера A2 × A2: h = 3 + 3 = 6

Строго доказано: `WA2A2_sem_direct_product : (36 * 2)%nat = 72%nat`

#### 3.2. Aut(A4) → SU(5) ГУТ

- W(A4) = S₅, |W(A4)| = 120 = 5!
- |Aut(A4)| = 240 = 2 · 120 (внешний автоморфизм Z₂)
- A4 живёт внутри H4 как подсистема ранга 4

Строго доказано: `Aut_A4_order_correct : Aut_A4_order = (2 * W_A4_order)%nat`

#### 3.3. Индексы вложения и нормировка ГУТ

В стандартной нормировке SU(5)-ГУТ, при проекции E8 → SU(5) → SU(3) × SU(2) × U(1):

- Индекс вложения U(1)_Y в SU(5): связан с нормировочным фактором 5/3 для g₁²
- Физическая гиперзарядная связь: g' = g₁ · √(3/5) (преобразование GUT-нормировки в SM-нормировку)

**ЧЕСТНО:** Строгое Coq-доказательство точной цепочки E8 → SU(3)×SU(2)×U(1) с индексами вложения в текущей базе кода **отсутствует**. Утверждения в `H4GaugeEmbedding.v` формализуют комбинаторные структуры (порядки групп Вейля), но не полную теорию представлений E8.

## 4. Калибровочные константы на масштабе объединения

### 4.1. Предполагаемое H4-происхождение

Из `H4GaugeEmbedding.v`, раздел 7, постулируется:

```
α₁⁻¹(M_GUT) = (h/2)  · (φ/2) = 15 · φ/2 ≈ 12.135
α₂⁻¹(M_GUT) = (h/3)  · (φ/2) = 10 · φ/2 ≈  8.090
α₃⁻¹(M_GUT) = (h/5)  · (φ/2) =  6 · φ/2 ≈  4.854
```

### 4.2. Отношения констант (строго доказано)

Строго доказано (Coq, `alpha_unification_ratios`):

```coq
Lemma alpha_unification_ratios :
  alpha1_inv_unified / alpha2_inv_unified = 3 / 2 /\
  alpha2_inv_unified / alpha3_inv_unified = 5 / 3.
Proof. ... field; auto. Qed.
```

Таким образом:
```
α₁⁻¹ : α₂⁻¹ : α₃⁻¹ = 15 : 10 : 6  (строго)
```

Эти отношения совпадают с предсказаниями SU(5)-ГУТ при нормировке Дженгинса-Гирарди-Ровелли.

### 4.3. G03 = h/10 = 3 (строго)

В `H4Derivations.v`:

```coq

Proof. replace (30/10) with 3 by field. ... Qed.
```

G03 = 3 = h/10 соответствует числу цветовых зарядов SU(3)_C.

## 5. Нормировка ГУТ и разрыв до физической 1/α(m_Z)

### 5.1. Что нужно для физической 1/α

Физическая обратная постоянная тонкой структуры при m_Z:

```
1/α(m_Z) ≈ 128.9
```

Из G01 (Trinity-формула):

```
G01 = 36 · φ · e² / π ≈ 137.07  → 1/α(0) (Томсоновский предел, α при q²=0)
```

Разница между 1/α(0) ≈ 137.036 и 1/α(m_Z) ≈ 128.9 обусловлена электромагнитным пробегом через пороги адронов и лептонов. Trinity-формула G01 воспроизводит **Томсоновский** предел 1/α(0), а не 1/α(m_Z).

**ЧЕСТНО:** Связь G01 = 36φe²/π с числом Кокстера h=30 **не является строгим выводом**. Коэффициент 36 = 6² = (h/5)² можно интерпретировать как квадрат делителя h на 5 (число SU(5)), но это post-hoc объяснение числового совпадения.

### 5.2. GUT-нормировка и суммарное условие

В SU(5)-ГУТ при нормировке по Джилибио:

```
1/α₁ + 1/α₂ + 1/α₃ = 1/α_s (при M_GUT, с учётом нормировки)
```

Из H4-предсказания:

```
α₁⁻¹ + α₂⁻¹ + α₃⁻¹ = (15 + 10 + 6) · φ/2 = 31 · φ/2 ≈ 25.08
```

Физическое значение при M_GUT (из пробега SM): ≈ 1/α_s(M_GUT) ≈ 1/24 ≈ 0.042 → 1/α_GUT ≈ 24. Расхождение в фактор ~25.08 vs 24 составляет ~4%.

**ЧЕСТНО:** Эта дискrepancy документирована в `RGRunning.v`:

> «The current definition `alpha_inv_at_mZ` does NOT yield the physical 1/alpha(m_Z) ~ 128. It is a sum of two GUT-normalized inverse-coupling-squared values each ~ O(0.1), so their sum is also ~ O(0.1), not ~128.»

Теоремы `alpha_from_H4` и `alpha_s_from_H4` в `RGRunning.v` остаются **Admitted** по этой физической причине.

### 5.3. Преобразование g₁ → g' (гиперзарядная связь)

В GUT-нормировке SU(5):
```
g₁²(GUT) = (5/3) · g'²(SM)
```

Это преобразование требуется для восстановления физической 1/α(m_Z) из GUT-переменных:

```
1/α(m_Z) = (5/3) · 1/g₁²(m_Z) + 1/g₂²(m_Z)
```

Эта конверсия **не реализована** в текущем `RGRunning.v`, что и является причиной Admitted-статуса `alpha_from_H4`.

## 6. Происхождение G02 из корневой системы H4/E8

### 6.1. Формула G02 = (√5 − 2)/2

Запишем через φ:
```
√5 = 2φ − 1
G02 = (2φ − 1 − 2)/2 = (2φ − 3)/2 = φ − 3/2
```

Связь с H4: число √5 является базовым иррациональным числом, порождающим φ. Корневая система H4 определяется именно над полем Q(√5) — это единственная конечная группа отражений с таким свойством (строго доказано: `phi_irrational_over_Q` в `H4GaugeEmbedding.v`).

**ЧЕСТНО:** Утверждение "G02 выводится из H4" является **численным соответствием**, а не аналитическим выводом из первых принципов. Значение α_s(m_Z) ≈ 0.118 определяется пробегом КХД от Λ_QCD ~ 200 МэВ до m_Z ~ 91 ГэВ через пертурбативную КХД с 3-5 петлями. Связь этого с √5 является наблюдением, а не объяснением.

### 6.2. G03 → α_s через h/10

G03 = h/10 = 3 = dim(SU(3)_C)/... — это число цветов. Обратная константа связи КХД при объединении:

```
α₃⁻¹(M_GUT) = 6 · φ/2 ≈ 4.854
```

Пробег вниз от M_GUT до m_Z (через однопетлевое УРГ с b₃ = -7):

```
α₃⁻¹(m_Z) = α₃⁻¹(M_GUT) + (b₃/4π²) · ln(Λ_H4/m_Z)
           ≈ 4.854 + (-7/4π²) · 32.6 ≈ 4.854 − 5.81 ≈ −0.96 ???
```

Это показывает, что наивная однопетлевая формула с H4-граничным условием даёт **отрицательный** α₃⁻¹(m_Z), что физически бессмысленно. Это именно та проблема, о которой честно сообщает `RGRunning.v` через `alpha_run_window`:

```coq
Axiom alpha_run_window :
  alpha_i_inv m_Z b2 > 0 /\ alpha_i_inv m_Z b3 > 0.
```

## 9. Итог: что на самом деле доказано

**Строго (математически):** H4 — уникальная группа отражений с числом Кокстера h=30=2×3×5, что обеспечивает тройное деление, соответствующее числам 2, 3, 5 — рангам калибровочных групп SU(2), SU(3), SU(5).

**Численно (без строгого вывода):** Формулы G01=36φe²/π и G02=(√5−2)/2 воспроизводят 1/α(0)≈137 и α_s(m_Z)≈0.118 с точностью 0.03−0.1%. Связь этих чисел с H4-структурой остаётся **феноменологическим наблюдением**, а не аналитическим выводом.

**Честный разрыв:** Для строгого вывода от G01, G02 до физических констант необходимо:
1. Полная теория представлений E8 → SM (цепочка ветвления с индексами)
2. Двухпетлевое пробег УРГ с пороговыми поправками
3. Преобразование GUT-нормировки g₁ → g' (фактор √(5/3))
4. Физическое объяснение коэффициентов 36, (√5-2)/2 из первых принципов E8/H4
