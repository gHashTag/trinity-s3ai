## 1. Введение

Согласно формулировке Коннеса (1996, теорема реконструкции; Чамседдин–Коннес 2007,
arXiv:0706.3688), **вещественная спектральная тройка** (A, H, D; J, γ) — это набор:

- **A** — унитальная \*-алгебра, представленная ограниченными операторами на H
- **H** — сепарабельное гильбертово пространство
- **D** — самосопряжённый оператор с компактной резольвентой (оператор Дирака)
- **J** — антилинейная изометрия (вещественная структура, «зарядное сопряжение»)
- **γ** — унитарный самосопряжённый оператор грейдирования (γ² = 1, в чётном случае)

удовлетворяющий **7 аксиомам**, перечисленным ниже.

**Физическая постановка для H4/600-клетка:**

| Компонент | Реализация в Trinity S3AI |
|-----------|--------------------------|
| A | ℂ[2I] (функции на вершинах 600-клетки) / предполагаемая подалгебра ℂ ⊕ ℍ ⊕ M₃(ℂ) |
| H | ℂ¹²⁰ ⊗ ℂ² ≅ ℂ²⁴⁰ (вершинное пространство ⊗ спиноры) |
| D | Граф-Дирак 600-клетки с икосианными коэффициентами (строится в Волне 8.1) |
| J | Антилинейное кватернионное сопряжение q ↦ q̄ |
| γ | Вещественный диагональный знаковый оператор (±1 на ±-вершинах) |

**Ключевое замечание:** Проверяется спектральная тройка над **конечным** дискретным
пространством (120 вершин). В конечномерном случае многие аксиомы выполняются
автоматически или тривиально.

## 2. Аксиома 1: Размерность (KO-dim)

### 2.1 Точная математическая формулировка

KO-размерность n (mod 8) определяется тройкой знаков (ε, ε', ε''):

```
J²        = ε · Id         (ε ∈ {+1, −1})
JD        = ε' · DJ        (ε' ∈ {+1, −1})
Jγ        = ε'' · γJ       (ε'' ∈ {+1, −1}, только в чётном случае)
```

Для Стандартной Модели (Чамседдин–Коннес 2007) требуется **KO-dim F = 6 (mod 8)**,
чтобы произведение M × F имело KO-dim = 4 + 6 = 10 ≡ 2 (mod 8).

Таблица знаков Коннеса (чётные n):

| n | ε = J² | ε' | ε'' |
|---|--------|-----|------|
| 0 | +1 | +1 | +1 |
| 2 | −1 | +1 | +1 |
| 4 | −1 | +1 | +1 |
| 6 | **+1** | **+1** | **+1** |

Для n = 6 требуется **(ε, ε', ε'') = (+1, +1, +1)**.

### 2.2 Статус доказательства

**ЧАСТИЧНО ДОКАЗАНО** — с честной оговоркой об амбигуитете n=0 vs n=6.

**Что доказано (Qed) в KODimension.v, Волна 5.1:**
- Тройка знаков: (ε, ε', ε'') = (+1, +1, +1) вычислена и верифицирована
- Соответствие таблице Коннеса при n=6: `connes_table_ko6` (Qed)
- Совместимость с n=6: `cell600_consistent_with_ko6` (Qed)
- Совместимость с n=0 (честное дополнение): `cell600_consistent_with_ko0` (Qed)
- Требование СМ выполнено: `cell600_satisfies_SM_KO_requirement` (Qed)

**Что остаётся открытым:**
- Тройка (+1,+1,+1) совместима с **обоими** n=0 и n=6
- Разграничение n=0 от n=6 требует доказательства внедиагональности J
  (формально: J перемешивает H⁺ ⊕ H⁻ — леворучную и праворучную части)
- Это помечено как `PHYSICAL_AXIOM: cell600_J_off_diagonal` в KODimension.v

**Вердикт:** **ЧАСТИЧНО** (знаки доказаны; однозначность KO-dim=6 от KO-dim=0 — PHYSICAL_AXIOM)

## 3. Аксиома 2: Регулярность

### 3.1 Точная математическая формулировка

Определим оператор δ = [|D|, ·] (коммутатор с |D| = √(D²)).
Аксиома регулярности требует:

```
для всех a ∈ A:  a, [D, a] ∈ ⋂_{k≥0} Dom(δ^k)
```

Это «гладкость» алгебры относительно Дирака: все элементы алгебры и их
«производные» [D,a] принадлежат всем областям определения итерированного δ.

### 3.2 Статус доказательства

**ТРИВИАЛЬНО ВЫПОЛНЕНО** в конечномерном случае.

**Аргумент:** Для конечного дискретного пространства (120 вершин):
- H = ℂ²⁴⁰ конечномерно
- D — матрица 240×240, все операторы ограничены
- |D| — тоже матрица, bounded
- Dom(δ^k) = всё H для всех k (ограниченные операторы не имеют проблем с областью)
- Следовательно: ⋂_{k≥0} Dom(δ^k) = H

**Нет специальных усилий не требуется:** В конечномерной спектральной тройке
аксиома регулярности выполнена автоматически для любой алгебры ограниченных операторов.

**Ссылка:** Эта тривиальность хорошо известна; см. Gracia-Bondia, Varilly, Figueroa
«Elements of Noncommutative Geometry», Birkhauser 2001, Section 9.3.

**Что доказано в Coq (SpectralTripleAxioms.v, Волна 8.2):**
- `axiom_regularity_finite_dim` (Qed) — формальное подтверждение тривиальности
  через принцип: в конечномерном H любой ограниченный оператор δ сохраняет всё H

**Вердикт:** **ВЫПОЛНЕНО** (тривиально в конечном измерении; Qed)

## 4. Аксиома 3: Конечность

### 4.1 Точная математическая формулировка

Гладкое гильбертово пространство:
```
H^∞ := ⋂_{k≥0} Dom(D^k)
```
должно быть конечно порождённым проективным A-модулем.

Иными словами: H^∞ ≅ e·A^N для некоторого идемпотента e ∈ M_N(A) и числа N.

### 4.2 Статус доказательства

**ТРИВИАЛЬНО ВЫПОЛНЕНО** в конечномерном случае.

**Аргумент:**
- H = ℂ²⁴⁰ конечномерно → Dom(D^k) = H для всех k
- H^∞ = H = ℂ²⁴⁰
- Любое конечномерное пространство является конечно порождённым свободным
  (следовательно, проективным) A-модулем над ℂ
- Для A = ℂ[2I] ≅ ℂ¹²⁰: H ≅ A² как A-модуль (приближённо, до разбиения)

**Что доказано в Coq (SpectralTripleAxioms.v, Волна 8.2):**
- `axiom_finiteness_trivial` (Qed) — конечномерное пространство само себе является
  конечно порождённым проективным модулем (тривиально)
- `H_infinity_eq_H` (Qed) — в конечном измерении Dom(D^n) = H для всех n

**Вердикт:** **ВЫПОЛНЕНО** (тривиально в конечном измерении; Qed)

## 5. Аксиома 4: Реальность (структура J)

### 5.1 Точная математическая формулировка

Оператор J (вещественная структура) должен быть антилинейной изометрией с:

```
J² = ε                          (ε из таблицы KO-dim)
JDJ⁻¹ = ε' · D                 (или эквивалентно: JD = ε' DJ)
JγJ⁻¹ = ε'' · γ                (или: Jγ = ε'' γJ)
[a, JbJ⁻¹] = 0   для всех a,b ∈ A   (правостороннее действие)
```

Последнее условие обеспечивает, что правое A-действие через J коммутирует с левым.

### 5.2 Статус доказательства

**ДОКАЗАНО** для знаков; **ЧАСТИЧНО** для структурного условия.

**Что доказано в Coq:**
- Знаки J² = +1, JD = +DJ, Jγ = +γJ: KODimension.v (9 Qed, Волна 5.1)
- Мультипликативность нормы кватернионов: QuaternionicLinearity.v (15 Qed, Волна 5.2)
- Изометричность кватернионного сопряжения: `right_mult_preserves_norm` (Qed)
- Коммутативность левого и правого умножения: `quaternion_full_associativity` (Qed)
- Условие [a, JbJ⁻¹] = 0 косвенно следует из `left_right_mult_commute_0` и т.д.

**Что остаётся открытым:**
- Полное доказательство условия [a, JbJ⁻¹] = 0 в матричной формулировке
  для всей алгебры A = ℂ[2I] (не только для кватернионной подалгебры)
- Внедиагональная структура J (уже упомянута в Аксиоме 1 как PHYSICAL_AXIOM)

**Вердикт:** **ЧАСТИЧНО** (знаки и J-изометрия: Qed; [a,JbJ⁻¹]=0: частично через PHYSICAL_AXIOM)

## 6. Аксиома 5: Первый порядок

### 6.1 Точная математическая формулировка

Оператор Дирака D удовлетворяет **условию первого порядка**:

```
[[D, a], JbJ⁻¹] = 0    для всех a, b ∈ A
```

Это означает, что D — «дифференциальный оператор первого порядка» в НКГ-смысле:
его коммутаторы [D, a] с элементами алгебры коммутируют с правым A-действием.

### 6.2 Статус доказательства

**ОТКРЫТО** — это главное препятствие для «настоящей NCG» в Trinity S3AI.

**Почему это трудно:**
Для непрерывных спектральных троек (гладкие многообразия) это условие проверяется
напрямую из локальности оператора Дирака. Для дискретного D на 600-клетке:

1. Граф-Дирак D зависит от выбора весов на рёбрах
2. Условие [[D,a], JbJ⁻¹] = 0 должно выполняться для ВСЕХ a,b ∈ A = ℂ[2I]
3. Алгебра ℂ[2I] имеет 120 образующих
4. В НКГ-смысле это условие отбирает «правильные» операторы D среди всех
   самосопряжённых операторов на H

**Текущее состояние в Trinity S3AI:**
- Оператор D на 600-клетке строится в Волне 8.1 (параллельно)
- Условие первого порядка для предложенного D **ещё не проверено**
- Это является главным ограничением: без этого условия нельзя утверждать,
  что мы имеем дело с «настоящей NCG» в смысле Коннеса

**Исторический контекст:**
Для Стандартной Модели в NCG (Чамседдин–Коннес, A_F = ℂ ⊕ ℍ ⊕ M₃(ℂ)):
- Условие первого порядка выполнено по конструкции (D_F явно построен)
- Для H4/600-клетка аналогичной конструкции пока нет

**В Coq (SpectralTripleAxioms.v):**
- Помечено как `Axiom axiom_first_order` с тегом `MATH_TODO`
- Это **единственная аксиома**, которая является подлинным математическим вопросом
  (не тривиальной в конечном случае и не разрешённой физическим аргументом)

**Вердикт:** **ОТКРЫТО** — тег `MATH_TODO` (ключевая незакрытая проблема NCG)

## 7. Аксиома 6: Ориентация (хиральный цикл)

### 7.1 Точная математическая формулировка

Существует **канонический цикл Хохшильда** степени n (= KO-dim):

```
γ = Σ_i a_i [D, b_i^(1)] · [D, b_i^(2)] · ... · [D, b_i^(n)]
```

такой, что γ² = 1, {D, γ} = 0 (антикоммутирует с D) и Jγ = ε'' γJ.

В чётном случае (n=6 чётное): γ — это оператор грейдирования (киральности),
аналог γ₅ в физике. Цикл Хохшильда реализует класс ориентации.

### 7.2 Статус доказательства

**ЧАСТИЧНО** — структура γ установлена; Хохшильдова реализация — OPEN.

**Что доказано:**
- В KODimension.v: ε'' = +1 (Jγ = +γJ), следовательно, γ и J согласованы
- В ChiralityAnalysis.v (Волна 6): формальная структура χ (аналог γ)
  определена через разбиение 120 вершин на ±-классы
- `naive_spectrum_is_vector_like` (Qed): вычислены размеры спинорного содержания
- Оператор γ в SpectralTripleAxioms.v определён аксиоматически (Аксиома)

**Что остаётся открытым:**
- Явная Хохшильдова реализация γ как цикла степени 6
  (требует полностью построенного D из Волны 8.1)
- Антикоммутирование {D, γ} = 0 требует конкретного D
- Связь с η-инвариантом S³/2I (открытый вопрос из ChiralityAnalysis.v)
- Проблема хиральности (вектороподобный спектр — см. Волна 6) означает,
  что физически γ может давать нулевой индекс Дирака

**Вердикт:** **ЧАСТИЧНО** (γ как оператор определён; Хохшильдов цикл — MATH_TODO)

## 8. Аксиома 7: Двойственность Пуанкаре

### 8.1 Точная математическая формулировка

**KR-теоретическая (K-теоретическая) двойственность:** Пересечение K-теории алгебры
A с K-гомологиями (определяемыми тройкой (A,H,D)) невырождено:

```
χ: K*(A) × K*(A) → ℤ
χ([e], [f]) = Index(D · e · f^op)  (инвариант Кассини)
```

Спаривание χ должно быть невырожденным в смысле: если χ([e], [f]) = 0 для всех [f],
то [e] = 0 в K*(A).

### 8.2 Статус доказательства

**ЧАСТИЧНО** — для конечного пространства тривиализуется; физически не закрыто.

**Аргумент упрощения (конечный случай):**

Для конечного дискретного пространства с A = ℂ[2I]:
- K₀(A) = K₀(ℂ[2I]) ≅ ℤ⁹ (по числу неприводимых представлений 2I)
- K₁(A) = 0 (для конечных групповых алгебр над ℂ)
- Спаривание χ в конечном случае сводится к невырожденности пересечения
  проекторов в A-модулях с учётом оператора D

**Что доказано в Coq:**
- Число неприводимых представлений 2I: 9 (ChiralityAnalysis.v, `rho_dims`)
- Сумма квадратов размерностей = 120 = |2I|: `sum_sq_rho_dims_eq_2I_order` (Qed)
- Для формального Coq-доказательства невырожденности спаривания требуется
  полноценная K-теория в Coq, что выходит за рамки текущего фреймворка

**Честная оценка:**
Для конечного пространства двойственность Пуанкаре, как правило, выполнена
(автоматически через тождество Бернштейна–Гельфанда–Гельфанда для группового
кольца конечной группы). Однако строгое доказательство в NCG-смысле требует
явной конструкции D и проверки невырожденности.

**Вердикт:** **ЧАСТИЧНО** (упрощается в конечном случае; строгая формализация — MATH_TODO)

**Дополнительные открытые вопросы:**
- Хиральность: спектр вектороподобен (Wave 6), механизм хиральности неизвестен

- Алгебра A: ℂ[2I] ≠ ℂ ⊕ ℍ ⊕ M₃(ℂ) без дополнительного аргумента

### Приоритетный путь к «настоящей NCG»

1. **Построить D** (Волна 8.1) с явными кватернионными весами
2. **Проверить условие первого порядка** [[D,a], JbJ⁻¹] = 0 для полного D
3. **Вычислить η-инвариант** S³/2I (путь к хиральности через eta)
4. **Доказать KO-dim = 6** (а не 0) через явную внедиагональность J
