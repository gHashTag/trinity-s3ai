## 1. Математический контекст

### 1.1 Вершины 600-ячейника как единичные кватернионы

600-ячейник — правильный четырёхмерный многогранник со 120 вершинами, лежащими на трёхмерной сфере S³. Ключевой факт (Conway–Smith, 2003):

**Теорема (Conway–Smith).** 120 вершин 600-ячейника образуют **группу** относительно квотернионного умножения. Эта группа изоморфна **бинарной икосаэдральной группе** 2I порядка 120.

Явно: если отождествить S³ ⊂ ℝ⁴ с единичными кватернионами {q ∈ ℍ : |q| = 1}, то

```
2I = { ±1, ±i, ±j, ±k,
       (±1 ± i ± j ± k)/2,
       все перестановки (0, ±1, ±φ, ±1/φ)/2 }
```

где φ = (1+√5)/2 — золотое сечение, 1/φ = φ − 1 = (√5−1)/2.

Это даёт |2I| = 8 + 16 + 96 = **120 элементов**.

### 1.2 Алгебраическая структура

- 2I ≅ SL(2, 𝔽₅) — специальная линейная группа над полем из 5 элементов
- 2I — двойное накрытие группы вращений икосаэдра: существует точная последовательность
  ```
  1 → {±1} → 2I → A₅ → 1
  ```
  где A₅ ≅ PSL(2, 𝔽₅) — знакочередующаяся группа на 5 элементах (группа вращений икосаэдра)
- 2I — **совершенная группа**: [2I, 2I] = 2I (тривиальная абелевизация)
- 2I вкладывается в SU(2) ≅ S³ как конечная подгруппа — это один из трёх случаев ADE-классификации конечных подгрупп SU(2):

| Группа | Порядок | Диаграмма Дынкина |
|---|---|---|
| Бинарная тетраэдральная 2T | 24 | E₆ |
| Бинарная октаэдральная 2O | 48 | E₇ |
| **Бинарная икосаэдральная 2I** | **120** | **E₈** |

### 1.3 Кольцо икосиан и решётка E₈

Целые линейные комбинации элементов 2I образуют **кольцо икосиан** 𝕀 ⊂ ℍ. Это кольцо, рассматриваемое над ℚ(√5), является максимальным порядком в ℍ ⊗ ℚ(√5). При отождествлении элементов 𝕀 с восьмикомпонентными рациональными векторами и соответствующем переопределении нормы получается **решётка E₈** в ℝ⁸.

## 2. Генераторы 2I

### 2.1 Стандартные генераторы

Группа 2I порождается двумя элементами:

**Генератор r** (поворот порядка 5):
```
r = (−1 + φ·i + (1/φ)·j)/2 = (−1 + φi + (φ−1)j)/2
```
с условием r⁵ = −1 (в 2I), r¹⁰ = 1.

**Генератор s** (поворот порядка 3):
```
s = (1 + i + j)/√3  [в нормировке поворота]
```
или в координатах 2I:
```
s = (−1 + i + j + k)/2
```
с условием s³ = −1.

**Соотношение:** (r·s)² = −1, что означает (r·s)⁴ = 1.

Презентация 2I как конечной группы:
```
⟨r, s | r^5 = s^3 = (rs)^2 = −1, r^10 = s^6 = (rs)^4 = 1⟩
```

### 2.2 Связь генераторов с φ

Ключевое наблюдение: в координатах генератора r явно присутствует φ:
```
r = (−1 + φ·i + (1/φ)·j)/2
```

Норма: |r|² = (1 + φ² + (1/φ)²)/4 = (1 + (φ+1) + (φ−1)²... 

Вычислим: φ² = φ+1, (1/φ)² = (φ−1)² = φ²−2φ+1 = 2−φ.  
Итого: |r|² = (1 + φ+1 + 2−φ)/4 = 4/4 = 1. ✓

Таким образом, φ здесь не нумерология, а **необходимый компонент** единичного кватерниона с икосаэдральной симметрией.

### 2.3 Число Кокстера и φ

Число Кокстера группы H₄ равно h = 30. Собственные значения элемента Кокстера группы W(H₄) суть:
```
e^{2πi·m/30}, m ∈ {1, 11, 19, 29}
```

Отсюда: cos(π/5) = (φ)/2, cos(2π/5) = (φ−1)/2 — золотое сечение входит через тригонометрию пятикратной симметрии.

## 3. Квотернионная структура действия 2I

### 3.1 Правое умножение как представление

Группа 2I действует на H ≅ ℍ ≅ ℝ⁴ **правым умножением**:
```
ρ_R : 2I → SO(4),   g ↦ (q ↦ q·g)
```

Это действие:
1. **Сохраняет норму**: |q·g| = |q|·|g| = |q| (т.к. |g| = 1)
2. **Квотернионно-линейно слева**: ρ_R(g)(λ·q) = λ·(q·g) для λ ∈ ℍ
3. **Коммутирует с левым умножением**: если L_λ(q) = λq, то L_λ ∘ ρ_R(g) = ρ_R(g) ∘ L_λ

Свойство (3) — это ключевое наблюдение для NCG.

### 3.2 Алгебра 2I-инвариантных операторов

Рассмотрим алгебру End_{2I}(ℍ) эндоморфизмов ℍ (как правого 2I-модуля).

**Теорема (следствие теоремы Шура).** Эндоморфизмы правого 2I-модуля ℍ — это **левое умножение** на элементы ℍ:
```
End_{2I}(ℍ) ≅ ℍ^{op} ≅ ℍ
```

Здесь мы используем изоморфизм ℍ^{op} ≅ ℍ через сопряжение (кватернионная алгебра самосопряжена как алгебра над ℝ).

Следствие: **алгебра 2I-инвариантных операторов на ℍ содержит копию ℍ** — именно как алгебра левых умножений.

### 3.3 Спектральный тройственный аспект

В конструкции Конна спектрального тройства (A, H, D):
- **A** — коммутативная алгебра пространства-времени (или конечная алгебра A_F)
- **H** — гильбертово пространство
- **D** — оператор Дирака

В случае 600-ячейника:
- Вершинное пространство V ≅ ℝ^{120} несёт действие 2I
- Пространство квотернионных функций на 2I содержит ℍ как алгебру
- Инвариантные операторы относительно 2I дают подалгебру ≅ ℍ

## 4. Связь с компонентой ℍ в A_F

### 4.1 Аксиоматика Конна

Алгебра конечного тройства A_F = ℂ⊕ℍ⊕M₃(ℂ) выводится Конном из набора аксиом (Connes 1996, "Gravity coupled with matter and the foundation of non-commutative geometry"):

1. **Вещественная структура** — существование антилинейного оператора J
2. **Порядок нуля** — [a, Jb*J⁻¹] = 0
3. **Порядок единицы** — [[D,a], Jb*J⁻¹] = 0
4. **Квотернионная линейность** — A содержит ℍ как прямое слагаемое

Условие квотернионной линейности физически мотивировано: ℍ-компонента отвечает за **слабое взаимодействие** (SU(2)_L — внутренний автоморфизм ℍ).

### 4.2 Как 2I мотивирует ℍ ⊂ A_F

Цепочка рассуждений:

```
600-ячейник (120 вершин на S³)
         ↓
  2I = группа вершин под квотернионным умножением (|2I| = 120)
         ↓
  Действие правым умножением на ℍ: ρ_R : 2I → SO(4)
         ↓
  End_{2I}(ℍ) ≅ ℍ (алгебра инвариантных операторов)
         ↓
  ℍ ⊂ A_F = ℂ⊕ℍ⊕M₃(ℂ) [связь с аксиоматикой]
```

**Важно:** Эта цепочка является **мотивационной**, а не строгим выводом A_F. 2I объясняет, **почему** ℍ появляется как алгебра симметрий действия группы вершин 600-ячейника. Но полный вывод A_F требует дополнительных аксиом Конна (прежде всего, условий на оператор Дирака и классификацию допустимых алгебр через K-теорию).

### 4.3 Роль φ в квотернионной структуре

Элементы 2I, явно содержащие φ:
```
(±1 ± φi ± (1/φ)j)/2  и перестановки координат
(±1/φ ± i ± φj)/2     и перестановки координат
(±φ ± (1/φ)i ± j)/2   и перестановки координат
```

Таких элементов 96 (три класса по 32). Они и обеспечивают **пятикратную симметрию** 2I, отличающую её от других конечных подгрупп SU(2).

Алгебраическая идентичность, лежащая в основе:
```
φ² = φ + 1,    φ + 1/φ = √5,    φ · (1/φ) = 1
```

## 5. Честная оценка: вывод или мотивация?

### 5.1 Что строго доказано математически

| Утверждение | Статус |
|---|---|
| 120 вершин 600-ячейника = 2I как подгруппа S³ | ✅ Строго (Conway–Smith 2003) |
| |2I| = 120 | ✅ Строго |
| 2I ≅ SL(2, 𝔽₅) | ✅ Строго |
| 2I ⊂ SU(2), ADE-соответствие с E₈ | ✅ Строго (McKay 1980) |
| End_{2I}(ℍ) ≅ ℍ | ✅ Строго (теорема Шура) |
| Генератор 2I явно содержит φ | ✅ Строго (прямые вычисления) |
| Φ в числе Кокстера h=30 группы H₄ | ✅ Строго |

### 5.2 Что является мотивацией, но не выводом

| Утверждение | Честная оценка |
|---|---|
| 2I → ℍ ⊂ A_F (полный вывод) | ❌ Мотивация, не вывод |
| A_F = ℂ⊕ℍ⊕M₃(ℂ) следует из 600-ячейника | ❌ Нужны аксиомы NCG |
| φ в генераторах 2I → φ в физических константах | ❌ Разрыв логической цепи |
| M₃(ℂ) из 2I (цветная часть) | ❌ Требует отдельного анализа |

### 5.3 Что нужно для полного вывода

1. **Конструкция спектрального тройства** из вершинного пространства 600-ячейника с оператором Дирака
2. **Аксиоматическая проверка**: все 7 аксиом Конна для полученного тройства
3. **Реконструкция A_F** через теорему Конна о реконструкции коммутативной геометрии — для конечного тройства это требует классификации допустимых алгебр

Отсутствие шагов 1–3 означает, что **квотернионная линейность 2I является реальным математическим фактом**, но **вывод A_F из 600-ячейника через 2I остаётся незавершённой программой**, а не доказанной теоремой.

### 5.4 Сравнение с другими подходами

В отличие от чистой нумерологии (например, «масса электрона ≈ φ^k»), здесь есть **структурная причина**: 

- 2I — это подлинная группа вершин 600-ячейника
- ℍ появляется как алгебра инвариантных операторов, а не произвольно
- Связь E₈ ↔ 2I ↔ ℍ поддержана теоремой Маккея

Это делает данный подход **наиболее сильным кандидатом** в проекте Trinity на содержательный вывод компоненты ℍ. Однако дистанция от «мотивации» до «строгого вывода A_F» остаётся значительной.

## 6. Coq-формализация

Файл `QuaternionicLinearity.v` формализует следующие результаты:

1. **Основные φ-тождества** для координат генераторов 2I:
   - |r|² = 1 (единичность генератора)
   - Координатные тождества через φ² = φ+1

2. **Порядки генераторов** (по модулю ±1 в SU(2)):
   - r¹⁰ = 1 (порядок 10, что соответствует порядку 5 в A₅)
   - s⁶ = 1 (порядок 6, что соответствует порядку 3 в A₅)

3. **Квотернионная линейность**:
   - Коммутирование правого умножения с левым в ℍ
   - Инвариантность нормы под действием 2I

4. **Связь с числом 120**:
   - |H₄| = 120² = 14400 (через φ-алгебру)
   - Размер 2I = 120 (аксиоматически)
