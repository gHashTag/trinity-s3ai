GOLDEN BRIDGE — Build the Bridge!

Pick one BLUE Data tile and one PURPLE Geometry tile. Good matches light bridge stones; falsified red tiles crack the bridge. A hypothesis-discovery puzzle, not a proven Theory of Everything.

How to play (3 steps)

  1. Click a BLUE Data tile (left tower) — e.g. "Higgs Mass".
  2. Click a PURPLE Geometry tile (right tower) — e.g. "Weak Force".
  3. Watch the central bridge stones light up gold. Light four stones to make the bridge stable. Avoid red ⚠ tiles — they crack the bridge!

Buttons:  Reset clears the board · Hint suggests the next best tile · Auto-build runs the bridge-builder for you · Try Recipe loads a starter set.  Tile edge colour: green/gold = safe, blue = empirical, amber = open conjecture, red = falsified.

Honesty notice. A high "bridge strength" only means the board passes more of the illustrative consistency checks than alternatives. Every tile is gated by a claim status; the worst such status is rendered next to the total. A falsified tile collapses the bridge regardless of the rest.

How to enable the interactive build 
cd games/trinity_fold
wasm-pack build crates/ring4_canvas --target web --features wasm \
  --out-dir ../../web/canvas/pkg

# Regenerate the static snapshot:
cargo run -p ring4_canvas --example snapshot_svg -- \
  web/canvas/snapshot.svg
