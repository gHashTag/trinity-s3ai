## Философия

Каждый заблокированный путь — это **постоянный актив**. Научное сообщество не должно повторять одни и те же ошибки. Этот TECH TREE строится как карта **исключения**: каждый узел либо `PROVEN` (доказан), либо `FALSIFIED` (опровергнут), либо `OPEN` (открыт). Нефальсифицированные узлы не считаются доказанными.

```
Принцип Васильевой:
  Негативный результат = постоянный актив научного сообщества
  Скорость = медленно + тщательно
  Метод   = взаимоисключение недоказанных путей
  Цель    = математический корень всего
```

## Уровни уверенности

| Символ | Статус | Описание |
|--------|--------|----------|
| ✅ **PROVEN** | Доказан | Coq `Qed.` или независимая верификация |
| ❌ **FALSIFIED** | Опровергнут | Формальное опровержение или экспериментальное исключение |
| 🔬 **EMPIRICAL** | Эмпирический | Числовое совпадение без вывода из принципов |
| 🔄 **OPEN** | Открыт | Не доказан и не опровергнут |
| ⛔ **BLOCKED** | Заблокирован | NGT-теорема закрывает путь |
| 📋 **PREREGISTERED** | Предзарегистрирован | Предсказание зафиксировано до данных |

## Методология: Принципы Васильевой

### Принцип 1: Негативный результат = Актив

Каждый ❌ FALSIFIED узел:
- Экономит время БУДУЩИХ исследователей
- Сужает пространство гипотез
- Является **вкладом в науку** наравне с положительным результатом

### Принцип 2: Медленно и Тщательно

```
НЕТ: быстро написать 130 формул и назвать это теорией
ДА:  медленно формализовать 1 формулу в Coq и доказать
```

### Принцип 3: Взаимоисключение Путей

Прежде чем начать новый путь — проверить:
1. Противоречит ли он NGT1–NGT4?
2. Есть ли прецедент провала?
3. Какой эксперимент его опровергнет?

### Принцип 4: Математический Корень

> "Я верю, что математический корень есть у всего. Мы будем идти медленно, взаимоисключая недоказанные теории и пути." — Васильева

Каждое численное совпадение — **гипотеза**, не факт.  
Факт — только после `Qed.`

### Принцип 5: Pre-Registration

Все предсказания фиксируются **до** получения данных в `PREDICTIONS_PREREGISTERED.md` с:
- Точным значением
- Идентификатором Coq-теоремы (если есть)
- Критерием фальсификации (σ, эксперимент, год)
